How to use

Find the animation you want to use

Copy the CSS into your code from the movers.css file

Add the animation class (e.g .fadeIn) to the element you would like to animate in your code

Or

Clone the movers.css library from GitHub

Link movers.css to your HTML

Add the selected animation .class to your element

Tweak animation as required and enjoy!
